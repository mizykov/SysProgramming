# How to build:

Command in terminal: `cmake ./`, next build exe `make`

# How to launch:

./main input.txt original.txt sort.txt inverseSort.txt

where:
+ input.txt - your text.
+ original.txt - text in canonical order
+ sort.txt - direct sorted text
+ inverseSort.txt - text in reverse order
