# Square equation solver

---

C programming course at ISP RAS.

---

Necessary tools:

+ gcc
+ doxygen

---

# How to build:

The build is supported by Makefile:

> Build an exe file, tests, and documentation:
>> make full
> Build an exe file
>> make exeFile
> Build an documentation
>> make docs
> Build an tests
>> make tests

# How to launch:

> Launch program:
>> make runExeFile
> Launch tests
>> make runTests
