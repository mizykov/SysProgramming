# Square equation solver

---

C programming course at ISP RAS.

---

Necessary tools:

+ gcc
+ doxygen

---

# How to build:

The build is supported by Makefile:

> Build an exe file, tests, and documentation:
>> make full
>
> Build an exe file
>> make exeFile
>
> Build the documentation (which is located in the docs - index.html):
>> make docs
>
> Build tests
>> make tests

# How to launch:

> Launch program:
>> make runExeFile
>
> Launch tests
>> make runTests
